# My Raycaster Game it is a simple raycasting and it makes a 3D world for you 

for running on linux you have to clone the repo and then write down make and after that you can write ./game and enjoy
git clone https://github.com/PouriaMashhadi/My_Raycaster_Launch.git
cd My_Raycaster_Launch/src
make
./game
